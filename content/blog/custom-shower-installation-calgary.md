## Steam Shower Systems

### What is a Steam Shower?
A steam shower creates a steam-filled environment for relaxation and therapeutic benefits. Popular features include:

- **Steam Generator**: $1,500 - $4,000
- **Steam Head**: $200 - $500
- **Control System**: $300 - $1,000
- **Proper Enclosure**: Required for steam retention

### Benefits of Steam Showers:
- **Relaxation**: Reduces stress and tension
- **Therapeutic**: Helps with muscle soreness
- **Skin Benefits**: Opens pores, improves circulation
- **Respiratory**: Can help with congestion
- **Luxury**: Ultimate bathroom experience

### Installation Requirements:
- Properly sealed enclosure
- Steam generator location (typically in nearby closet)
- Proper ventilation
- Quality materials that can handle moisture

## Custom Shower Installation Process

### 1. **Design & Planning** (1-2 weeks)
- Measure space
- Discuss preferences
- Create design
- Select materials
- Obtain permits if needed

### 2. **Demolition** (1-2 days)
- Remove existing shower
- Prepare space
- Check plumbing/electrical

### 3. **Plumbing & Electrical** (2-3 days)
- Install/update plumbing
- Electrical for lighting/steam
- Waterproofing preparation

### 4. **Waterproofing** (1-2 days)
- Critical step for preventing leaks
- Proper membrane installation
- Calgary building code compliance

### 5. **Tile Installation** (3-5 days)
- Install base
- Tile walls
- Grout and seal
- Quality craftsmanship

### 6. **Fixture Installation** (1-2 days)
- Install showerheads
- Install valves
- Install doors
- Final connections

### 7. **Finishing** (1 day)
- Caulking
- Final inspections
- Cleanup
- Walkthrough

**Total Timeline**: 3-5 weeks typically

## Maintenance Tips for Custom Showers

### Daily
- Squeegee after use
- Keep doors open to air out
- Wipe down surfaces

### Weekly
- Deep clean tiles
- Clean glass doors
- Check for any issues

### Monthly
- Inspect caulking
- Check grout condition
- Clean fixtures

### Annually
- Professional deep clean
- Re-seal natural stone if needed
- Check steam system (if applicable)
