# Public Assets Folder

This folder contains static assets that are served directly by Next.js.

## Logo Files

To add your new logo for the header/menu:

1. **Place your logo file here** in the `public` folder
2. **Recommended file names:**
   - `logo.png` (or `logo.svg` if you have an SVG version)
   - `logo-white.png` (if you need a white/light version)
   - `logo-dark.png` (if you need a dark version)

3. **Recommended dimensions:**
   - **Width**: 200-300px (or 400-600px for 2x retina)
   - **Height**: 50-60px (or 100-120px for 2x retina)
   - **Format**: PNG with transparent background (or SVG)

4. **After adding your logo**, the file will be accessible at:
   - `/logo.png` (if named `logo.png`)
   - `/logo.svg` (if named `logo.svg`)
